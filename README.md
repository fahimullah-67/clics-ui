#CLICS Complete Frontend
 - Banking Schemes Explorer

Add SchemesPage, SettingsPage, and WatchlistPage components with filtering and notification features

- Implemented SchemesPage for exploring and comparing banking schemes with filters for loan types, banks, and amount ranges.
- Added SettingsPage for managing user profile, preferences, security settings, and notifications.
- Created WatchlistPage to track favorite loan schemes with alert toggles and delete functionality.
- Configured TypeScript support with tsconfig.node.json.
- Set up Vite configuration for the project with path aliasing and Tailwind CSS integration.
- Updated package-lock.json for dependency management.
## Features

- **Schemes Explorer**: Browse and compare banking schemes with advanced filtering
- **Settings Management**: Configure user preferences and security settings
- **Watchlist Tracking**: Monitor favorite schemes with custom alerts

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Node.js
